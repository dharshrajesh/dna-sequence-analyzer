# dna-sequence-analyzer
## Overview
This project is a Python-based DNA sequence analysis tool.

## Features
- Calculates GC Content
- Generates Reverse Complement
- Validates DNA sequences
- Counts nucleotide frequency

## Technologies Used
- Python
- String processing techniques

## How to Run
1. Run the Python file.
2. Enter a DNA sequence containing A, T, C, G.
3. View analysis results.
